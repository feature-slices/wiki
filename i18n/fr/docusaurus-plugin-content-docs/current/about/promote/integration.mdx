# Aspects de l'intégration

**Avantages**:
- [Vue d'ensemble](/docs/get-started/overview#advantages)
- Revue de code
- Intégration des nouveaux

**Inconvénients**:
- Complexité mentale
- Barrière d'entrée élevée
- "Enfer des couches"
- Problèmes typiques des approches basées sur les fonctionnalités
